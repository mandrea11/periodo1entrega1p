PROMEDIO.SI

La función PROMEDIO.SI es una función estadística. Se usa para analizar el rango de datos y encontrar el valor medio de todos los números en un rango de celdas, basándose en un criterio específico.

COINCIDIR

Con esta funcion se puede realizar un dato que se encuentre en determinada posicion, muestra la fila o columna en la que se encuentre le valor buscado; para esto se usa la siguiente sintaxis:=COINCIDIR( "Valor buscado" en este espacio se pone la posicion de la celda en la cual se ingresa el vaolor que deseamos que coincida, "Matriz buscada", en este campo, se elige todo el rango que se busca examinar, por ultimo, "tipo de coincidencia" en este lucar se ingresa 0 o 1, con el 0 se busca una coincidencia exacta).
